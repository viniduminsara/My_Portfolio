Salon Management System

The JavaFX standalone application for a salon management system is a comprehensive software solution designed to streamline and automate various aspects of running a salon. It provides a user-friendly interface that allows salon owners, managers, and staff to efficiently manage their daily operations and enhance customer service.

Connect-four game

The Connect Four game developed with JavaFX incorporates an AI player, offering an engaging and challenging single-player experience. This version of the classic strategy game allows players to test their skills against an intelligent computer opponent.JavaFX enhances the gaming experience by providing a visually appealing and interactive interface. Players can make their moves by clicking on the desired column, and the AI player responds with its calculated move.

Chat Application

The chat application developed with Java socket programming is a versatile and efficient communication tool that allows users to exchange messages in real-time over a network. It leverages Java's socket programming capabilities to establish a reliable and secure connection between multiple clients. The chat application supports both text-based and multimedia-based messaging. Users can send and receive text messages, share files, and exchange images within the chat room.
